[블록 3] 정적→개발 이관·로직 연결 지시 (Codex용)
================================================================
첨부한 정적 HTML(_publish/<경로>)을 app/public/<동일경로>로 이관하고
로직을 연결해줘. CODEX_HANDOFF.md 와 07_api-spec.md 규칙을 따른다.
- _publish/ 원본은 절대 수정하지 말 것(단방향 이관).
- data-bind → API 응답 필드 연결(매핑표 기준).
- data-repeat → 배열 렌더링, 빈 배열이면 empty-result 노출.
- data-action → app/src/ 핸들러 연결.
- data-chart → Chart.js + 디자인가이드 색상.
- 예외 UI(hidden)는 조건부 토글만, 새 마크업 생성 금지.
- 기본 Mock 모드로 외부 비용 0 유지.
