# 04-2. 고급자 모드 와이어프레임

**파일 경로:** `_docs/04_wireframe/01_expert.md`
**선행 문서:** `04_wireframe/00_common-and-user.md`

## 공통 사항
고급자 전 단계도 초급자와 동일하게 좌측 입력 패널(60%) + 우측 실시간 프롬프트 패널(40%) 2분할을 사용한다. 다만 입력 패널은 아코디언·데이터 테이블 형태로 더 조밀하며, 단계 인디케이터는 5단계(●━○━○━○━○)로 표기한다. GNB·푸터·타이포·컬러 토큰은 5-1 표준을 그대로 적용한다.

---

```
│                               │                          │
│ [H3] 가치 우선순위 밸런스       │ [H3] 전문가 AI 요청서     │

│                               │ │ [CPU: AMD 필수],       │ │

```
