## 1. 공통 레이아웃 골격 (전 화면 통일)

모든 사용자 화면(랜딩·초급자·고급자)은 아래 4영역 골격을 공유한다. 관리자 화면은 GNB 대신 LNB(좌측 사이드바)를 쓰며 별도 정의한다.

```
┌──────────────────────────────────────────────────────────┐
│  [GNB] 전역 상단 내비게이션 (고정, 높이 64px)              │
├──────────────────────────────────────────────────────────┤
│  [페이지 헤더] 화면 제목 + 단계 인디케이터 (높이 가변)     │
├──────────────────────────────────────────────────────────┤
│                                                            │
│  [콘텐츠 영역] 화면별 본문                                  │
│                                                            │
├──────────────────────────────────────────────────────────┤
│  [푸터] 회사 정보·고객센터·정책 링크 (높이 가변)           │
└──────────────────────────────────────────────────────────┘
```

### 1.1 GNB (전역 상단 내비게이션) — 전 사용자 화면 동일
```
┌──────────────────────────────────────────────────────────┐
│ 🖥 팝콘PC AI    [초급자 모드] [고급자 모드]   로그인/회원가입 │
│ (로고·좌측)      (중앙 모드 전환 탭)        장바구니 마이페이지│
└──────────────────────────────────────────────────────────┘
```
좌측은 브랜드 로고(클릭 시 `index.html` 이동 + 상태 초기화), 중앙은 [초급자 모드]/[고급자 모드] 전환 탭(현재 위치 활성 표시), 우측은 [로그인/회원가입]·[장바구니]·[마이페이지/견적 보관함]을 배치한다. GNB는 스크롤 시에도 상단 고정(sticky)한다.

### 1.2 푸터 — 전 화면 동일
```
┌──────────────────────────────────────────────────────────┐
│ 팝콘PC | 고객센터 1234-5678 | 이용약관 · 개인정보처리방침   │
│ © 2026 팝콘PC. All rights reserved.                        │
└──────────────────────────────────────────────────────────┘
```

### 1.3 관리자 LNB (좌측 사이드바) — 관리자 화면 전용
```
┌──────────┐
│ 팝콘PC 관리 │
├──────────┤
│ ▸ 대시보드 │
│ ▸ 비용관제 │
│ ▸ 상품관리 │
│ ▸ 통계분석 │
├──────────┤
│ 관리자님 ▾ │
└──────────┘
```

## 2. 타이포그래피 위계 표준 (전 화면 통일)

콘텐츠가 시각적으로 명확히 구분되도록 글자 크기·굵기·용도를 아래와 같이 고정한다. 모든 화면은 이 토큰만 사용한다(임의 크기 금지).

| 토큰명 | 크기 | 굵기 | 용도 | 비고 |
|--------|------|------|------|------|
| Display | 32px | Bold(700) | 랜딩 히어로 메인 카피 | 화면당 1회 |
| H1 (제목) | 28px | Bold(700) | 페이지 대표 제목 | 화면당 1개 |
| H2 (부제목) | 22px | SemiBold(600) | 섹션 구분 제목 | |
| H3 (소제목) | 18px | SemiBold(600) | 카드/그룹 제목 | |
| Body-L (강조 본문) | 16px | Regular(400) | 주요 설명 | 행간 1.6 |
| Body (기본 본문) | 14px | Regular(400) | 일반 텍스트 | 행간 1.6 |
| Caption (보조) | 12px | Regular(400) | 도움말·각주·라벨 | 회색 |
| Button (버튼) | 15px | SemiBold(600) | 버튼 라벨 | |
| Number (수치 강조) | 24px | Bold(700) | 가격·점수·FPS 등 | Primary 색 |

폰트 패밀리는 한국어 가독성을 위해 `Pretendard`를 1순위, 미설치 시 `'Noto Sans KR', system-ui, sans-serif`로 폴백한다. 모든 화면 텍스트·라벨·버튼은 **한국어로 통일**하며, 기술 약어(CPU, NVMe 등)만 영문을 허용한다.

## 5. 초급자 모드 와이어프레임

초급자 전 단계는 좌측 입력 패널(60%)과 우측 실시간 프롬프트 패널(40%)의 2분할을 공유한다.

5-1 산출물이 완성되었습니다. 공통 골격(GNB·푸터·LNB), 타이포 위계 9종(Display 32px ~ Caption 12px), 컬러 토큰, 그리고 랜딩 + 초급자 6개 화면 와이어를 모두 한국어로 통일해 정의했습니다. 이 타이포·골격 표준은 다음 단계(디자인 가이드)에서 그대로 CSS 변수로 전환됩니다.
