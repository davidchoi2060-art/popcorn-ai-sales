# 04-3. 관리자 허브 & dev_index 게이트 와이어프레임

**파일 경로:** `_docs/04_wireframe/02_admin-and-gate.md`
**선행 문서:** `04_wireframe/00`, `01`

## 공통 사항
관리자 전 화면은 좌측 LNB(고정 폭 200px) + 우측 콘텐츠 영역 구조를 공유한다. 상단에는 간소화된 관리자 헤더(현재 메뉴 경로 + 관리자 계정)를 둔다. 푸터는 사용자 화면과 동일하되 간소화한다.

```
┌────────┬─────────────────────────────────────────────────┐
│ [LNB]  │ [관리자 헤더] 대시보드 > AI 비용관제   관리자님 ▾ │
│ 사이드 ├─────────────────────────────────────────────────┤
│  바    │ [콘텐츠 영역]                                    │
│ 200px  │                                                 │
└────────┴─────────────────────────────────────────────────┘
```

```
├──────────────────────────────────────────────────────────┤
│ [H2] ① 화면 구현 진척도 매트릭스                          │
│ ┌────────┬──────────────┬────────────┬──────────────────┐ │
│ │화면 ID │ 파일명       │ 구현 상태  │ 이동 링크        │ │
│ ├────────┼──────────────┼────────────┼──────────────────┤ │
│ │FR-LND  │ index.html   │●배포완료   │[미리보기][동작][프롬프트]│
│ │FR-BEG  │ step1-pur... │🟡코딩중    │[미리보기][동작][프롬프트]│
│ │ ...    │ ...          │⚪대기      │ ...              │ │

├──────────────────────────────────────────────────────────┤
│ [H2] ② 개발 테스트 도구 (개발자 전용)                     │
│ ┌────────────────────────┐ ┌────────────────────────┐    │

│ │[H3] 추천 가중치 룰셋     │ │[H3] 3사 비용 관제       │    │
│ │재고소진[──●────] 40%    │ │Gemini $12.5 ChatGPT $18.2│   │

└──────────────────────────────────────────────────────────┘
```

진척도 매트릭스의 각 행은 [정적 미리보기](`_publish/` 링크), [동작 테스트](`app` 또는 `localhost:3000` 링크), [수정 프롬프트](PROMPT_MASTER 해당 블록 복사) 3개 버튼을 제공한다. 기획자·고객·개발자가 각자 필요한 링크로 진척을 확인한다.

5-3 산출물이 완성되어 **5단계 와이어프레임이 모두 끝났습니다.** 정리하면, 5-1(공통 골격·타이포 표준 + 랜딩·초급자), 5-2(고급자), 5-3(관리자·게이트)으로 전체 27개 화면의 골격을 한국어 통일·타이포 위계 적용·GNB/LNB/헤더/푸터 통일로 완성했습니다.
