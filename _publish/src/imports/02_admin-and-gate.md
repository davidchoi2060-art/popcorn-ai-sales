## 0. V3.0 개요

관리자 허브는 V3.0부터 기존의 **AI 비용 관제 중심 백오피스**가 아니라, 실제 MD/운영자가 출근 후 먼저 확인하고 조치해야 하는 업무 순서에 맞춘 **비즈니스 운영 중심 백오피스**로 재정렬한다.

관리자 V3.0의 우선순위는 다음과 같다.

1. 운영자 메인 컨트롤 타워  
2. 상품 마스터 및 재고 제어  
3. 가격 정책 및 추천 가중치 제어  
4. 마케팅 및 사용자 행동 분석  
5. 시스템 인프라 및 개발 특수 제어  

기존 비용 관제, Rate Limit, Circuit Breaker는 관리자 첫 화면의 중심이 아니라, `adm-system-limit`에서 통합 관리한다.  
단, `adm-dashboard`에서는 AI 비용 상태를 **미니 위젯**으로 축소 노출한다.

### 1.1 LNB 메뉴 구성

| 메뉴 그룹 | 포함 Screen 키 | 기본 이동 | 우선순위 |
|----------|----------------|-----------|----------|
| 대시보드 | `adm-dashboard` | `adm-dashboard` | 1 |
| 상품/재고 | `adm-product-master`, `adm-csv-import` | `adm-product-master` | 2 |
| 가격/정책 | `adm-price-policy`, `adm-recommend-weights` | `adm-price-policy` | 3 |
| 마케팅분석 | `adm-keywords`, `adm-click-report`, `adm-funnel` | `adm-keywords` | 4 |
| 시스템제어 | `adm-system-limit`, `dev-hub` | `adm-system-limit` | 5 |

### 1.2 관리자 공통 규칙

- 한국어 UI를 기본으로 하며, 기술 약어만 영문을 허용한다.
- 폰트는 Pretendard / Noto Sans KR 기준이다.
- 본문 행간은 1.6을 유지한다.
- H1은 28px Bold, H2는 22px SemiBold, H3는 18px SemiBold를 사용한다.
- 수치 강조는 24px Bold + Primary `#0075d5`를 사용한다.
- 위험 상태는 `#D32F2F`, 정상 상태는 `#2e7d32`, 주의 상태는 `#f9a825`를 사용한다.
- 페이지 배경은 `#f7f9fc`, 카드 배경은 흰색, 테두리는 `#e0e0e0`을 사용한다.
- 모든 간격은 8px 그리드 기준으로 설계한다.
- 반응형 768px 이하에서는 LNB가 햄버거 메뉴로 접힌다.
- 개인정보, API Key, 세션 토큰은 화면에 평문 노출하지 않는다.
- 데이터 바인딩 대상은 `data-bind`, 반복 목록은 `data-repeat`, 버튼 액션은 `data-action`, 차트는 `<canvas data-chart>` 기준으로 표시한다.

`adm-dashboard`는 실시간 견적 피드, 특가 CTR, 키워드 트렌드, 가격동향, AI 비용 미니 위젯을 포함하는 운영자 메인 컨트롤 타워가 되며, `adm-product-master`는 상품 검색, AI 호환성 필드 수정, 품절/단종 1초 제외 토글을 담당하는 핵심 화면으로 격상되었습니다.

AI 비용, Rate Limit, Circuit Breaker는 `adm-system-limit`로 통합했으며, `dev-hub`는 V3.0 관리자 빠른 이동과 개발 테스트 도구를 포함하는 독립 게이트로 확장했습니다.
