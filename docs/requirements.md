**1. 프로젝트 개요 요구사항**

**1-1. 프로젝트 기본정의**

| **항목** | **내용** |
|----|----|
| 프로젝트명 | 팝콘PC AI 기반 조립PC 추천 및 구매 시스템 구축 |
| 구축 목적 | 사용자가 원하는 예산과 사용 목적에 맞추어 팝콘PC 보유 부품 기준으로 최적의 조립PC 구성을 추천하고, 비교/시각화/장바구니/구매까지 한 번에 진행할 수 있는 시스템 구축 |
| 핵심 목표 | 사용자의 선택 피로도를 줄이고, 구매 전환율을 높이며, 재고 및 마진을 고려한 판매 최적화 구현 |
| 주요 차별점 | 일반 쇼핑몰이 아닌, AI가 사용 목적/예산/성능 기준에 따라 조합을 자동 추천하는 맞춤형 조립PC 구매 시스템 |
| AI 경험 차별점 | 사용자가 선택한 조건을 AI에게 요청하는 자연어 프롬프트 형태로 자동 변환하여, 사용자가 AI가 자신의 요구를 어떻게 이해하는지 확인할 수 있도록 함 |
| 결과 시각화 차별점 | 추천된 PC의 특징, 성능, 호환성, 가격 구성, 재고, 업그레이드 가능성을 이미지·그래프·도표로 직관적으로 제공 |
| 대상 고객 | 게임용 PC 구매자, 사무용/가정용 PC 구매자, 영상편집/디자인 작업용 PC 구매자, 방송용/개발용/AI입문용 PC 구매자 |
| 운영 주체 | 팝콘PC |
| 추천 기준 | 성능, 가격, 호환성, 재고 보유 여부, 판매 우선순위, 마진 정책, 이벤트 상품 우선 노출 등 |

**1-2. 시스템 구축 목표**

| **구분** | **목표 내용** |
|----|----|
| 사용자 편의성 | 초보자도 몇 가지 옵션만 선택하면 쉽게 조립PC를 추천받을 수 있어야 함 |
| AI 이해도 제공 | 사용자가 선택한 조건이 AI에게 어떤 요청 문장으로 전달되는지 실시간으로 확인할 수 있어야 함 |
| 판매 효율성 | 팝콘PC가 보유한 재고와 판매 전략에 맞는 상품이 우선 추천되어야 함 |
| 자동화 | 수동 상담 없이도 기본적인 견적 추천이 자동으로 이루어져야 함 |
| 시각화 | 추천된 PC가 어떤 특징과 성능 수준을 가지는지 이미지, 그래프, 도표로 직관적으로 보여줘야 함 |
| 신뢰감 | 추천 이유, 호환성 검증 결과, 재고 상태, 출고 가능성을 명확하게 표시해야 함 |
| 구매 전환 | 추천 → 비교 → 옵션 변경 → 장바구니 → 결제까지 자연스럽게 연결되어야 함 |
| 관리 효율 | 관리자가 재고, 가격, 우선 판매 품목, 추천 룰 등을 쉽게 수정할 수 있어야 함 |

**2. 사용자 관점 요구사항**

**2-1. 사용자 유형 정의**

| **사용자 유형** | **설명** | **주요 니즈** |
|----|----|----|
| 초보 사용자 | PC 부품 지식이 부족한 고객 | 쉽게 추천받고 싶음 |
| 중급 사용자 | CPU/GPU 정도는 이해하는 고객 | 어느 정도 직접 선택 + AI 추천 받고 싶음 |
| 고급 사용자 | 세부 부품까지 비교 가능한 고객 | 상세 호환성/성능/확장성 확인 원함 |
| 재구매 고객 | 이전 구매 이력이 있는 고객 | 기존 구매 기록 기반 업그레이드 추천 원함 |
| 기업/단체 구매 고객 | 사무용 다량 구매 고객 | 목적별 일괄 견적, 예산별 다수 구성 원함 |

**2-2. 사용자가 입력하는 조건 요구사항**

| **항목** | **입력 방식** | **상세 내용** |
|----|----|----|
| 사용 목적 | 다중 선택/우선순위 선택 | 게임, 사무용, 인터넷/강의, 영상편집, 그래픽디자인, 3D작업, 방송, 개발, AI입문, 주식/멀티모니터 등 |
| 예산 | 직접 입력 + 구간 선택 | 예: 50만원 이하, 50~80만원, 80~120만원, 120~200만원, 200만원 이상 |
| 선호 CPU | 선택사항 | Intel, AMD, 상관없음 |
| 선호 GPU | 선택사항 | NVIDIA, AMD, 내장그래픽, 상관없음 |
| 메모리 용량 | 선택사항 | 8GB, 16GB, 32GB, 64GB 이상, 추천받기 |
| 저장장치 | 선택사항 | SSD 500GB, 1TB, 2TB 이상, HDD 추가 여부 |
| 게임 사양 | 게임명 선택 | 예: 배틀그라운드, 로스트아크, FC온라인, 디아블로, GTA, 발로란트 등 |
| 작업 프로그램 | 프로그램 선택 | 포토샵, 프리미어, 애프터이펙트, 오토캐드, 블렌더 등 |
| 해상도/모니터 환경 | 선택 | FHD, QHD, 4K, 듀얼모니터 이상 |
| 소음/발열 선호 | 선택 | 저소음 우선, 성능 우선, 균형형 |
| 케이스 디자인 | 선택 | RGB 선호, 무난한 디자인, 화이트, 블랙, 미들타워 등 |
| 업그레이드 여유 | 선택 | 추후 업그레이드 고려, 현재 가성비 우선 |
| 운영체제 포함 여부 | 선택 | Windows 포함, 미포함 |
| 브랜드 선호/제외 | 선택 | 특정 브랜드 선호 또는 제외 |
| 배송/조립 옵션 | 선택 | 기본 조립, 선정리, 테스트 리포트 포함 여부 |
| AI 요청 프롬프트 | 자동 생성 + 직접 수정 | 사용자가 선택한 사용 목적, 예산, 선호 조건을 기반으로 AI에게 요청하는 자연어 문장을 자동 생성한다. 사용자는 최종 확인 단계에서 해당 문장을 확인하고 필요 시 직접 수정할 수 있다. |
| AI 요청 프롬프트 기본 문구 | 시스템 자동 삽입 | “안녕하세요! 사용 목적과 예산에 맞는 조립PC 구성을 찾고 있습니다.” 문장을 기본 시작 문구로 사용한다. |
| AI 요청 프롬프트 확인 | 실시간 미리보기 | 조건 선택 화면 우측 패널에서 선택값이 변경될 때마다 프롬프트 문장이 실시간 업데이트되어야 한다. |
