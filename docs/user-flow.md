**설계 조건 요약**

| **항목** | **결정 내용** |
|----|----|
| 서비스 구조 | 독립 서비스 (별도 도메인) |
| 회원 시스템 | 기존 팝콘PC 홈페이지 회원과 연동 (SSO) |
| 조건 선택 방식 | 초보자/고급자 모드 분리 |
| AI 요청 방식 | 사용자가 선택한 조건을 기반으로 AI에게 요청하는 자연어 프롬프트를 자동 생성 |
| 조건 선택 화면 구조 | PC에서는 좌측 조건 선택 + 우측 AI 요청 프롬프트 미리보기 패널 구조 |
| AI 요청 기본 문구 | “안녕하세요! 사용 목적과 예산에 맞는 조립PC 구성을 찾고 있습니다.” |
| 구매 흐름 | 장바구니 담기 시점에 기존 홈페이지로 리다이렉트 |
| 추천 결과 표시 방식 | 가성비형/권장형/고성능형 3개 견적을 카드와 시각화 대시보드로 표시 |
| 추천 결과 시각화 | 대표 이미지, 부품 이미지, 성능 그래프, 호환성 검증, 가격 구성, 업그레이드 가능성 표시 |
| 관리자 접근 | /admin 경로 |
| 관리자 권한 | 3단계 (슈퍼관리자 / 상품담당자 / 주문담당자) |
