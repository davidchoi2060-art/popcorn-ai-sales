## 2. 공통 UI 컴포넌트 상세

### A. GNB (Global Navigation Bar - 헤더)
- **좌측:** 팝콘PC AI 추천 서비스 로고 (클릭 시 메인 랜딩페이지 `index.html` 이동)
- **중앙:** 모드 전환 탭 버튼 `[초보자 모드]` `[고급자 모드]` (활성화된 모드는 Primary 컬러 강조 테두리 적용)
- **우측:** 회원 서비스 링크 `[로그인/회원가입]` (SSO 연동 통합 팝업 호출), `[장바구니]`, `[마이페이지]`

### B. 푸터 (Footer)
- **구성 요소:** 카피라이트, 사업자 정보, 고객센터 안내, A/S 정책 바로가기 링크
- **중요 정책 고지 (필수 법적 안내):** > "본 시스템에서 제공하는 PC 성능 정보는 내부 기준 및 외부 벤치마크 기반의 예상치이며, 사용자의 실제 사용 환경 및 드라이버 버전에 따라 달라질 수 있습니다."

### C. 우측 AI 요청 프롬프트 패널 (SCREEN 02 공통 컴포넌트)
- **패널 타이틀:** "AI에게 이렇게 요청할게요"
- **기본 문구:** "안녕하세요! 사용 목적과 예산에 맞는 조립PC 구성을 찾고 있습니다. 아직 선택한 조건이 없습니다. 사용 목적과 예산을 선택하면 AI에게 요청할 문장이 자동으로 완성됩니다."
- **상태 뱃지:** `[자동 생성됨]` 또는 `[사용자 수정됨]` 라벨 표시
- **하단 제어 버튼:** `[복사]` `[직접 수정]` `[초기화]`
